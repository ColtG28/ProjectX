# ProjectX Downloads

This repository uses GitHub Releases for private-friendly distribution.

These links always point to the latest release assets:

- [Linux portable desktop bundle](https://github.com/ColtG-28/ProjectX/releases/latest/download/ProjectX-linux.tar.gz)
- [Linux portable desktop bundle SHA-256](https://github.com/ColtG-28/ProjectX/releases/latest/download/ProjectX-linux.tar.gz.sha256)
- [Windows portable app](https://github.com/ColtG-28/ProjectX/releases/latest/download/ProjectX-windows.zip)
- [Windows portable app SHA-256](https://github.com/ColtG-28/ProjectX/releases/latest/download/ProjectX-windows.zip.sha256)
- [macOS](https://github.com/ColtG-28/ProjectX/releases/latest/download/ProjectX-macos.dmg)
- [macOS SHA-256](https://github.com/ColtG-28/ProjectX/releases/latest/download/ProjectX-macos.dmg.sha256)

Notes:

- These links work without a public GitHub Pages site.
- For a private repository, viewers need repository access/authentication to download release assets.
- New tagged releases replace the "latest" target automatically, so this page does not need manual edits for each release.
- The macOS DMG contains a real `ProjectX.app` bundle. It is ad-hoc signed for local usability, but still unsigned/notarization-free in the Apple trust sense, so first launch may require right-click `Open` or approval in `Privacy & Security`.
- The Linux download is a portable bundle with a desktop file, icon assets, and `install.sh` rather than an AppImage in this branch.
